Ten projekt pokazuje, jak tworzyć użytkowników IAM i przypisywać im ograniczone uprawnienia w praktyce.

## Co zostało zrobione

- Przypisanie ról
- Testy dostępów
- Analiza Access Advisor

## Czego się nauczyłem
- Jak ograniczać dostęp do zasobów
- Jak testować uprawnienia
